Simple Selectors

Selector Name Example Explanation
Element Selector p {} Selects all <p> elements
Class Selector .class {} Selects all elements with class="class"
Class Selector
(with specified element tag)		 p.class {} Selects all <p> elements with class="class"
ID Selector #id {} Selects all elements with id="id"
Universal Selector * {} Selects all elements
Grouping Selector p, .class, #id {} Selects all <p> elements, elements with class="class", and elements with id="id"

Combinator Selectors

Selector Name Example Explanation
Descendent Selector div p {} Selects all <p> elements that are descendents of <div> elements
Child Selector div > p {} Selects all <p> elements that are children of <div> elements
General Sibling Selector div ~ p {} Selects all <p> elements that are preceded by a <div> element, where all elements have the same parent
Adjacent Sibling Selector div + p {} Selects the first <p> element after each <div> element, where both elements have the same parent

Psuedo-class Selectors

Selector Name Example Explanation
Active Selector a:active {} Selects the <a> element being pressed with the mouse
Visited Selector a:visited {} Selects all visited <a> elements
Link Selector a:link {} Selects all unvisited <a> elements
Checked Selector input:checked {} Selects all checked <input> elements
Enabled Selector input:Enabled {} Selects all enabled <input> elements
Disabled Selector input:disabled {} Selects all disabled <input> elements
Focus Selector input:focus {} Selects the <input> element that has focus
In-range Selector input:in-range {} Selects <input> elements with min and max attributes, that have a value within that specified range
Out-of-range Selector input:out-of-range {} Selects <input> elements with min and max attributes, that have a value outside that specified range
Valid Selector input::valid {} Selects <input> elements with valid values
Invalid Selector input:invalid {} Selects <input> elements with invalid values
Optional Selector input:optional {} Selects <input> elements with no "required" attribute
Read-only Selector input:read-only {} Selects <input> elements with a "readonly" attribute specified
Read-write Selector input:read-write {} Selects <input> elements with no "readonly" attribute
Lang Selector p:lang(ln) {} Selects <p> elements that have a lang attribute starting with "ln"
Hover Selector :hover {} Selects elements on mouse hover
Empty Selector :empty {} Selects elements with no children
First-child Selector :first-child {} Selects elements that are first children of their parent elements
Last-child Selector :last-child {} Selects elements that are last children of their parent elements
Nth-child Selector :nth-child(2) {} Selects elements that are the nth (in this case 2nd) children of their parent elements
Nth-last-child Selector :nth-last-child(2) {} Selects elements that are the nth (in this case 2nd) to last children of their parent elements
Only-child Selector :only-child {} Selects an element if it is the only child of its parent element
First-of-type Selector p:first-of-type {} Selects elements that are the first of type <p> in their parent elements
Last-of-type Selector p:last-of-type {} Selects elements that are the last of type <p> in their parent elements
Nth-of-type Selector p:nth-of-type(2) {} Selects elements that are the nth (in this case 2nd) of type <p> in their parent elements
Nth-last-of-type Selector p:nth-last-of-type(2) {} Selects elements that are the nth (in this case 2nd) to last of type <p> in their parent elements
Only-of-type Selector p:only-of-type {} Selects <p> element if it is the only <p> element of their parent
Not Selector :not(p) {} Selects all elements that are not <p> elements
Root Selector :root {} Selects the document's root element
Target Selector #id:target {} Selects the element with id="id" if the user arrives there from a hyperlink to that anchor name

Pseudo-element Selectors

Selector Name Example Explanation
First-letter Selector p::first-letter {} Selects the first letter of every <p> element
First-line Selector p::first-line {} Selects the first line of every <p> element
Marker Selector ol li::marker {} Selects the marker elements of all <li> elements that are children of <ol> elements
Selection Selector ::selection {} Selects the portion of an element that is selected by a user
After Selector ::after {content: url(img.img);} Inserts content (in this case, img.img) after each element
Before Selector ::before {content: url(img.img);} Inserts content (in this case, img.img) before each element

Attribute Selectors

Selector Name Example Explanation
[attribute] Selector [target] {} Selects elements with attribute "target" set
[attribute=value] Selector [target=_blank] {} Selects elements with attribute target="_blank"
[attribute~=value] Selector [title~=flower] {} Selects elements with attribute title containing "flower"
[attribute|=value] Selector [lang|=ln] {} Selects elements with attribute lang starting with "ln"
[attribute^=value] Selector a[href^="https"] {} Selects <a> elements with attribute href starting with "https"
[atrribute$=value] Selector a[href$=".pdf"] {} Selects <a> elements with attribute href ending with ".pdf"
[attribute*=value] Selector a[href*="pattern"] {} Selects <a> elements with attribute href containing "pattern"
